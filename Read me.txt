# Stock Analysis — AAPL, MSFT, TCS (2025)
# Relative Valuation - VISA, ICICI, SNOWFLAKE

## Objective
To analyse and compare the price performance, volatility, correlation, and 
risk-adjusted returns of Apple (AAPL), Microsoft (MSFT), and Tata Consultancy 
Services (TCS) over the 2025 calendar year using Python-based financial data analysis.

## What This Project Covers
- Data collection using yfinance API
- Price trend visualisation with 30-day and 50-day rolling averages
- Daily returns calculation and volatility comparison across stocks
- Correlation analysis between US and Indian market stocks
- Sharpe Ratio computation to evaluate risk-adjusted performance

## Tools & Libraries
- Python, Jupyter Notebook
- yfinance, pandas, numpy, matplotlib

## Key Findings
- TCS delivered the highest absolute price growth in 2025
- AAPL and MSFT showed moderate correlation, while TCS showed 
  near-zero correlation with both US stocks, reflecting differences in market 
  exposure and macroeconomic drivers
- Apple delivered the best risk-adjusted return (highest Sharpe Ratio) despite 
  Microsoft exhibiting higher absolute volatility

## How to Run
1. Clone the repo
2. Install dependencies: `pip install yfinance pandas numpy matplotlib`
3. Open `stock_analysis.ipynb` in Jupyter Notebook
