# Stock Analysis — Equity Research & Quantitative Finance

## Objective
To build a comprehensive equity research and quantitative finance portfolio
using Python — covering stock analysis, valuation, risk modelling,
algorithmic strategy testing, and factor models.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `stock_analysis.ipynb` | Comparative price trend, volatility, rolling averages, and Sharpe Ratio analysis for AAPL, MSFT, and TCS |
| `india_us_stock_comparison.ipynb` | Cross-sector, cross-border analysis of L&T, Infosys, and JPMorgan — correlation, returns, and risk-adjusted performance |
| `relative_valuation.ipynb` | P/E, P/B, and EV/EBITDA ratio analysis for Visa, ICICI Bank, and Snowflake with market interpretation |
| `Monte_Carlo_Simulation.ipynb` | 1,000-path Geometric Brownian Motion simulation for AAPL — 5th/50th/95th percentile price outcomes over 252 trading days |
| `Backtesting.ipynb` | 20/50-day moving average crossover strategy backtested against buy-and-hold on 2 years of AAPL data |
| `CAPM_&_Beta.ipynb` | Beta calculation, CAPM expected return, and Alpha generation analysis for Blackstone, Accenture, and Cognizant |

## Key Concepts Covered
- Daily returns, volatility, rolling averages, Sharpe Ratio
- Correlation matrix, covariance matrix, portfolio construction
- Relative valuation — P/E, P/B, EV/EBITDA
- Monte Carlo simulation using Geometric Brownian Motion
- Strategy backtesting with signal generation and cumulative return comparison
- CAPM, Beta, Alpha — systematic vs unsystematic risk

## Tools & Libraries
- Python, Jupyter Notebook
- pandas, numpy, matplotlib, seaborn, yfinance
